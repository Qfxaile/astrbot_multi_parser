# 更新日志

仅记录重要的用户可见变化；完整细节请查看 Git 提交历史。

## v1.0.0 - 2026-07-23

### 新增

- 支持 AstrBot 管理员在私聊中通过 B站二维码登录，并自动保存或清除插件 Cookie。
- 支持 AstrBot 管理员在私聊中通过百度二维码登录贴吧；遇到人机或设备验证时明确终止。
- 支持 AstrBot 管理员在私聊中通过微博二维码登录，仅保存解析所需的最小 Cookie；遇到人机、设备验证或不受信任的登录跳转时明确终止。
- 支持 AstrBot 管理员通过微信二维码授权腾讯元宝，并仅保存视频号短链解析所需的最小 Cookie。
- 支持 AstrBot 管理员在私聊中通过小黑盒官网委托的微信二维码登录，并仅保存解析所需 Cookie。
- 新增平台登录状态查询和进行中登录取消命令。

### 变更

- 将所有平台 Cookies 集中到配置页面的 `cookies` 分组。

### 说明

- 抖音、小红书和知乎二维码登录获取 Cookies 暂未实现，当前调用会登录失败，需要在配置页面手动填写。
